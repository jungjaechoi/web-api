개념 이해하기

-> 종이증명서를 O2O(Online to Offline) 검증을 통해 전자 문서화 하고자 함.

1. NFT 기술의 장점

NFT 기술방식을 적용하여 이미지 검증에 활용함

  • NFT는 블록체인으로써 출처, 발행시간 및 획수, 소유자 내역 등의 추적이 쉬움
  • NFT는 위조가 어려워 희소성을 더 잘 보장할 수 있다.

2. 블록체인의 특성

  • 블록체인의 특징인 무결성과불가역성으로 문서의 고유식별값을 블록체인에 저장
  • 문서의 존재 여부, 진위 여부, 원본 여부 등의 확인 가능

3. 응용서비스 및 기대효과

  • 종이증명서는 스마트폰이 없는 모든 연령대가 사용가능
  • 종이증명서 일차적인 증명 수단 (스마트폰 보완적)
  • 해외여행, 국내여행시 종이증명서 활용 필요
  • KeyPoint가 저장되면 각 경로 생성 버튼을 클릭하여 경로 생성 화면으로 전환합니다.
  • 백신여권 미채택 국가에 종이증명서로 증빙 효과
  • 아프리카, 아시아 국가들 중에서 종이증명서 만 채택하는 국가와의 연동성
  • KeyPoint가 저장되면 각 경로 생성 버튼을 클릭하여 경로 생성 화면으로 전환합니다.
  • 종이증명서 - 관계기관 발급 증명서류(백신접종증명서)
  • 백신여권 - 서버 기반 백신접종 신원증명 방식
